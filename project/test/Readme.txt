Требования:
1)Прежде всего данную директорию следует разместить вместе с template в одной директории

2)В template должна храниться директория data с обучающей выборкой в data/multiclass/train

Описание данной директории(краткое):
data - здесь будут храниться данные для обучения и для теста, здесь расположен файл whole_labels.txt, содержащий в себе размеченную выборку

src - python-скрипты: (качество кода не проверялось)
	average.py - вычисляет среднюю Precision в output.txt
	gendata.py - разбивает выборку на обучающую и тестовую случайным образом в заданном отношении
	my_compare - вычисляет Precision

tmp - здесь будут расположены файлы с моделью model.txt и с предсказаниями predictions.txt

Makefile - файл, с помощью которого можно сделать следующее:
	Набрав в консоли, находясь в данной директории:
		make all: будет запущена серия тестов
			результат: файл output.txt со значениями Precision на каждой итерации
		make build: соберётся проект
		make clean: проект будет очищен от созданных в make build исполняемых и объектных файлов
		make train: будет обучен SVM классификатор
		make predict: будет протестирован SVM классификатор
		make test: будет замерена Precision на тесте
		make cross: будет запущена серия тестов
			результат: файл output.txt со значениями Precision на каждой итерации
